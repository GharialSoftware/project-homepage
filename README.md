# project-homepage

## Build 

Node.js version used: `v20.18.1`

We recommend the use of `nvm`

```
nvm install 20.18.1
nvm use 20.18.1
```

### Install Hugo 

Hugo version used: `v0.143.1`

Windows:

```
winget install Hugo.Hugo.Extended
```

### Install dependancies 

```
cd ./src/
npm install
```

### Run dev server 

```
npm run dev
```

### Build site

```
npm run build
```
